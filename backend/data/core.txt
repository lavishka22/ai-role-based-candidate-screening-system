Machine learning interviews should assess data understanding, feature engineering, model selection, validation, overfitting, underfitting, bias-variance tradeoff, metrics, deployment, and monitoring. For classification, candidates should understand precision, recall, F1 score, ROC-AUC, confusion matrices, class imbalance, threshold tuning, and error analysis. For deep learning, candidates should understand neural networks, CNNs, RNNs, transformers, optimization, regularization, dropout, batch normalization, and transfer learning. For retrieval augmented generation, candidates should understand document ingestion, chunking, embeddings, vector databases, similarity search, prompt grounding, hallucination reduction, citations, and evaluation.
